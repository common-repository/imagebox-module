== Description ==

Images is the important part of the website. 

The Beaver Builder Imagebox plugin is a single module of Beaver Builder modules that can be simply dragged, dropped and customized as you want. it's additional style of imagebox.


Check <a href="https://beaverdemo.flickdevs.com/image-box/" target="_blank">Demo</a>

Feel free to <a href="https://flickdevs.com/contact-us/" target="_blank">contact us</a> if you find there are no beaver builder add-on/modules available that match your requirements. We are available to create it for you.
 
Note : This plugin is a module of Beaver Builder Plugin.(https://wordpress.org/plugins/beaver-builder-lite-version/) and will only work with Beaver Builder plugin installed and active.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress
